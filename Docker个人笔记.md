# Docker个人笔记

本文主要是个人对Docker的一些学习总结


## Docker简介

Docker是一个开源的引擎，可以轻松的为任何应用创建一个轻量级的、可移植的、自给自足的容器。开发者在笔记本上编译测试通过的容器可以批量地在生产环境中部署，包括VMs（虚拟机）、bare metal、OpenStack 集群和其他的基础应用平台。

### Docker 架构详解

Docker 的核心组件包括：
    - Docker 客户端 - Client
    - Docker 服务器 - Docker daemon
    - Docker 镜像 - Image
    - Registry 镜像仓库
    - Docker 容器 - Container
